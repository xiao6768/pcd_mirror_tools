# pcd_mirror_tools

This tool convert Carla HD map from Left-handed coordinate to right-handed coordinate for Lidar-localization use , for example Autoware.AI ...etc.



## usage 

python pcd_mirror_tools.py mode input_pcd output_mirrow_pcd 
mode : 0 y mirror
     : 1 x mirror


## pre-install package
```
pip install pypcd
```
